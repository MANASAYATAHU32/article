PROGRAM PERLINDUNGAN KEANEKARAGAMAN HAYATI:

PT PLN Indonesia Power Unit Pembangkitan PLTD/G Tello telah melakukan rogram perlindungan keanekaragaman hayati yang sudah diimplementasikan pada tahun 2020-2024


Program Anggaran

Secara umum terjadi peningkatan terhadap nilai indeks keanekaragaman hayati di seluruh area konservasi PT. PLN IP UBP Tello UP PLTD/G Tello. Semakin baiknya upaya perlindungan keanekaragaman hayati PT. PLN IP UBP Tello UP PLTD/G Tello dapat diketahui dari grafik peningkatan nilai absolut program perlindungan keanekaragaman hayati
